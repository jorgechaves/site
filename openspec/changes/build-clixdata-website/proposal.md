## Why

ClixData Dados & Infra is a data services company (database administration, BI dashboard development, ETL/ELT, and data infrastructure administration) with no web presence. A modern, interactive marketing website is needed to present the company's services, build credibility, and capture qualified leads — with a primary emphasis on BI dashboard and data visualization work using SQL and Python.

## What Changes

- Introduce a brand-new public-facing marketing website for ClixData Dados & Infra.
- Add a **Hero / landing section** communicating the company value proposition with a clear primary call-to-action (contact / request a quote).
- Add a **Services section** detailing the four core offerings: database administration, BI dashboard development (highlighted as the flagship service), ETL/ELT, and data infrastructure administration.
- Add an **About / Expertise section** highlighting SQL and Python as core competencies.
- Add a **Contact section** with a lead-capture form (name, email, message) and company contact details.
- Apply a **modern, dynamic, interactive design** using a Facebook/LinkedIn-inspired blue color palette, responsive layout, smooth scrolling, and subtle motion/animation.
- Establish the **project tech stack and build tooling** for a static, deployable single-page site.

## Capabilities

### New Capabilities
- `marketing-site`: The overall public website structure, navigation, layout, responsive behavior, and visual/brand design system (Facebook/LinkedIn blue palette).
- `services-showcase`: Presentation of ClixData's service offerings with BI/dashboard development emphasized, plus the SQL/Python expertise highlight.
- `lead-capture`: Contact section with a validated lead-capture form and company contact information.
