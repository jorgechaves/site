## ADDED Requirements

### Requirement: Core services presentation
The website SHALL present ClixData's four core services as distinct cards or blocks: Database Administration, BI Dashboard Development, ETL/ELT, and Data Infrastructure Administration. Each service SHALL include a title, a short description, and an icon or visual.

#### Scenario: All services listed
- **WHEN** a visitor views the Services section
- **THEN** four service blocks are displayed: Database Administration, BI Dashboard Development, ETL/ELT, and Data Infrastructure Administration
- **AND** each block shows a title, description, and icon

### Requirement: BI dashboard development emphasis
The website SHALL visually emphasize BI dashboard and business-intelligence development as the company's flagship service (e.g., featured placement, larger card, highlighted styling, or a dedicated highlight area).

#### Scenario: Flagship service highlighted
- **WHEN** a visitor views the Services section
- **THEN** the BI Dashboard Development offering is visually distinguished from the other services as the primary/flagship offering

### Requirement: SQL and Python expertise highlight
The website SHALL highlight SQL and Python as core technical competencies used to deliver data and BI solutions, in an About/Expertise area.

#### Scenario: Tech competencies shown
- **WHEN** a visitor views the About/Expertise section
- **THEN** SQL and Python are presented as key technologies/skills the company uses
